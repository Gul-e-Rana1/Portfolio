# Gul-e-Rana Portfolio

This repository contains a personal portfolio website built to showcase Gul-e-Rana's profile, skills, projects, and contact information.

## About

The portfolio is a responsive frontend website with a modern UI, animated sections, and reusable React components.

## What's Included

- Hero section
- Skills and technologies
- Projects showcase
- Process/experience sections
- Contact footer
- Responsive design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Motion
- Lucide React

## Run Locally

```bash
pnpm install
npm run dev
```

Open the site in your browser:

```text
http://localhost:5173/
```
